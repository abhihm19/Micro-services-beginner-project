# Micro-services-biginner-project

##Pre-Requisites
1. Java 1.8
2. Maven 3+
3. Any suitable IDE

##Procedure
1. Clone the project to suitable location
2. Open IDE and extract each services as existing maven project.
3. Run the services in the following order
   a. Service-Registry
   b. User-service
   c. Concern-service
   d. Config-server
   e. API-Gateway
